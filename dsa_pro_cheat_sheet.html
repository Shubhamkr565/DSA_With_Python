🪟 Sliding Window

Use when: subarray/substring + condition on a window of elements
max subarraymin windowlongest substringk-size windowcontiguous
Fixed Window:
1
Build first window of size k
2
Slide: add right element, remove left element
3
Track result at each valid window
left = 0; result = 0; window_sum = 0 for right in range(n): window_sum += arr[right] if right >= k-1: result = max(result, window_sum) window_sum -= arr[left]; left++
Variable Window:
1
Expand right until condition is violated
2
Shrink left until condition is valid again
3
Update answer at every valid window
Time: O(n) Space: O(1) or O(k)
⚠ Mistake: moving left more than 1 step at a time in variable window

👉👈 Two Pointer

Use when: sorted array + pair/triplet condition, or palindrome check
two sum sorted3sumpalindromecontainer waterremove duplicates

Opposite Ends

left=0, right=n-1
Move based on condition
Used in: pair sum, container

Same Direction

slow & fast pointer
slow writes, fast reads
Used in: remove dups, merge

# Two Sum in sorted array l, r = 0, n-1 while l < r: s = arr[l] + arr[r] if s == target: return [l,r] elif s < target: l++ else: r--
Time: O(n) Space: O(1)
⚠ Only works on SORTED arrays (for opposite-end variant)

🔍 Binary Search

Use when: sorted data OR monotonic function (if X works, X+1 also works)
sorted arrayfind minimumfirst/last positionsearch spacerotated
1
Define: lo, hi, mid = lo+(hi-lo)//2
2
Ask: is mid a valid answer? shrink accordingly
3
Always exit with lo == hi (the answer)
while lo < hi: mid = lo + (hi - lo) // 2 if feasible(mid): hi = mid # search left else: lo = mid + 1 # search right return lo
Binary Search on Answer Template:
"Find minimum X such that condition(X) is True"
→ lo = min_possible, hi = max_possible
→ if condition(mid): hi=mid else: lo=mid+1
Time: O(log n) Space: O(1)
⚠ mid overflow: use lo + (hi-lo)//2, NOT (lo+hi)//2

➕ Prefix Sum & Hashing

Use when: subarray sum, frequency count, seen-before lookups
subarray sum = krange querycount pairsanagramtwo sum
Prefix Sum:
prefix[0] = 0 for i in range(n): prefix[i+1] = prefix[i] + arr[i] # sum(i,j) = prefix[j+1] - prefix[i]
HashMap pattern (subarray sum = k):
seen = {0:1}; total=count=0 for x in arr: total += x count += seen.get(total-k, 0) seen[total] = seen.get(total,0)+1
Time: O(n)Space: O(n)
⚠ Initialize seen={0:1} — the empty subarray has sum 0

🔗 Linked List Tricks

Use dummy node always — prevents null pointer edge cases!

Fast & Slow (Floyd)

slow moves 1, fast moves 2
Meet = cycle exists
Find middle, find cycle start

Dummy Node

dummy → head
Never lose the head!
return dummy.next

# Find middle slow = fast = head while fast and fast.next: slow = slow.next fast = fast.next.next # slow is now middle
Reverse Linked List:
prev, cur = None, head while cur: nxt = cur.next cur.next = prev prev = cur; cur = nxt return prev
Time: O(n)Space: O(1)

📚 Stack & Monotonic Stack

Use when: next greater/smaller element, valid parentheses, histogram
next greaternext smallerbracketshistogramdaily tempsstock span

Mono Increasing

Pop when: current < top
→ Next Smaller Element
Stack stays increasing ↑

Mono Decreasing

Pop when: current > top
→ Next Greater Element
Stack stays decreasing ↓

# Next Greater Element st, res = [], [-1]*n for i, v in enumerate(arr): while st and arr[st[-1]] < v: res[st.pop()] = v st.append(i)
⚠ Stack stores INDICES not values (most of the time)
Time: O(n)Space: O(n)

🌳 Trees: BFS & DFS

BFS — Level Order

Use queue (deque)
Process level by level
Shortest path in unweighted

DFS — Recursion

Preorder/Inorder/Post
Use call stack implicitly
Path problems, subtree

# BFS template from collections import deque q = deque([root]); res = [] while q: level = [] for _ in range(len(q)): node = q.popleft() level.append(node.val) if node.left: q.append(node.left) if node.right: q.append(node.right) res.append(level)
# DFS tree template def dfs(node): if not node: return 0 left = dfs(node.left) right = dfs(node.right) # process current node return 1 + max(left, right)
BFS = shortest path. DFS = path existence / exhaustive search
Time: O(n)Space: O(h) DFS / O(w) BFS

🧩 Dynamic Programming

Use when: optimal substructure + overlapping subproblems
max/min wayscount pathsknapsacksubsequencepartition
DP Types Cheatsheet:
TypeStateFamous Problems
1D DPdp[i]Climbing stairs, house robber
2D DPdp[i][j]LCS, Edit distance, Grid paths
Knapsackdp[i][w]0/1 Knapsack, subset sum
Intervaldp[l][r]Matrix chain, burst balloons
Tree DPdp[node]Max path sum, house robber III
Bitmaskdp[mask]TSP, assignment problem
The 5-Step DP Framework:
1
Define dp[i] clearly in plain English
2
Base case: smallest input answer
3
Recurrence: dp[i] = f(dp[i-1], dp[i-2]...)
4
Order: fill in correct order (usually L→R)
5
Answer: dp[n] or max(dp)
# LCS — 2D DP dp = [[0]*(m+1) for _ in range(n+1)] for i in range(1,n+1): for j in range(1,m+1): if s1[i-1]==s2[j-1]: dp[i][j]=dp[i-1][j-1]+1 else: dp[i][j]=max(dp[i-1][j], dp[i][j-1])
⚠ Space optimize: 2D → 1D when only prev row is needed
Time: O(n²) typicalSpace: O(n) optimized

⛏ Heap & Greedy

Heap: when you repeatedly need min/max. Greedy: locally optimal = globally optimal

Min Heap

heapq in Python (default)
heappush / heappop
Top-K smallest elements

Max Heap

Push negatives in Python
heappush(h, -val)
Top-K largest elements

# K largest elements import heapq heap = [] for x in nums: heapq.heappush(heap, x) if len(heap) > k: heapq.heappop(heap) # heap contains k largest
Greedy Checklist:
✓
Can I make a locally optimal choice at each step?
✓
Does local optimum always lead to global optimum?
✓
Famous: interval scheduling, jump game, gas station
Heap ops: O(log n)Top-K: O(n log k)

🔁 Backtracking

Use when: find ALL solutions, explore all paths, permutations/combinations
all combinationspermutationssubsetsN-Queensword searchsudoku
Universal Backtracking Template:
def backtrack(state, choices): if is_solution(state): result.append(state.copy()) return for choice in choices: if is_valid(choice): make_choice(choice) # add backtrack(state, ...) undo_choice(choice) # remove ← KEY!
3 Questions before backtracking:
1. What is my choice at each step?
2. What is my base case (solution found)?
3. What do I undo after recursion?
Time: O(n! or 2ⁿ)Space: O(n) stack

🌐 Trie & Union Find

Trie

Word insert/search: O(L)
Prefix matching
Each node = dict of children

Union Find (DSU)

Group elements together
find() with path compression
union() by rank

# Trie Node class Trie: def __init__(self): self.children = {}; self.end = False def insert(self, word): cur = self for c in word: cur = cur.children.setdefault(c, Trie()) cur.end = True
# Union Find parent = list(range(n)) def find(x): if parent[x]!=x: parent[x]=find(parent[x]) return parent[x] def union(x,y): parent[find(x)] = find(y)
Trie: O(L)DSU: O(α(n)) ≈ O(1)
